Wimbus Studios API - v.25

Equipment Route:

https://wimbusstudios.herokuapp.com/equip

String Query Description Type
equipName Name of the equipment. String
equipChar Name of character that can use the equipment. Choose from: - Ian, Xenon, Bera, or Everyone String
equipLoc Indicates the location of where the equipment can be found. Choose from: Kami, Magik, or Ornery Coast String
equipDesc Equipment description. String

Example:

https://wimbusstudios.herokuapp.com/equip
?equipChar=Ian&equipLoc=Haven

Item Route:

https://wimbusstudios.herokuapp.com/items

String Query Description Type
itemName Name of the item. String
itemChar Name of the character that can use the item. Choose from: Ian, Xenon, Bera, or Everyone String
itemLoc Indicates the location of where the item can be found. Choose from: Kami, Magik, or Ornery Coast String
itemDesc Equipment description. String

Example:

https://wimbusstudios.herokuapp.com/items
?itemChar=Everyone&itemLoc=Ornery Coast

Beastiary Route:

https://wimbusstudios.herokuapp.com/beastiary

String Query Description Type
enemyName Name of the enemy. String
enemyURL URL where enemy image is stored. String
enemyLoc Indicates the location of where the enemy can be found. Choose from: Haven, Ornery Coast, or Beaks Peaks String
enemyType Type or enemy. Choose from: Boss or Enemy String

Example:

https://wimbusstudios.herokuapp.com/beastiary
?enemyLoc=Beaks Peaks&equipDesc=Boss
